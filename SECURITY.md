# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in BeatOps, please report it responsibly.

**Email:** security@beatops.io

Please include:
- Description of the vulnerability
- Steps to reproduce
- Affected version(s)

We will respond within 48 hours and work with you to resolve the issue before public disclosure.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| Older   | No        |
